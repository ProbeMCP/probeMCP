# Support

ProbeMCP is currently pre-release.

## Community Support

Use GitHub issues for:

- reproducible bugs
- backend adapter requests
- documentation gaps
- feature proposals
- QEMU or OpenOCD validation reports

## Security Issues

Use GitHub security advisories when available. Avoid posting exploit details,
unsafe command sequences, or target-damaging workflows in public issues.

## Commercial Support

Commercial support is not available yet. The project roadmap leaves room for
future paid support, certified backend profiles, and lab/debugging workflow
services.
